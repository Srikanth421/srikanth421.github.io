About This Project

Teaching AI and Cloud Systems Through Visual Intuition

🎯 Teaching Philosophy

Every concept is presented two ways: a precise technical explanation with formulas and worked examples on the left, and an everyday analogy with visual intuition on the right. Students who love math get rigor. Students who prefer stories get clarity. Everyone learns.

🛠 How to Use

These pages are designed for projector/classroom display. Open any lecture in a browser — all animations trigger on scroll, all sections are navigable via the sticky top bar, and everything works offline with zero dependencies. Download and teach.

📁 Self-Contained HTML

Each lecture is a single HTML file with embedded CSS and JavaScript. No frameworks, no build tools, no npm install. Save the file and open it — it works everywhere, forever.
