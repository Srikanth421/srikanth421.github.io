The 4 Formulas — Memorise These

Formula 01
Sigmoid Function
Squashes any real number → probability in (0, 1)
σ(z) = 1 / (1 + e^(-z)) σ(-∞) → 0 σ(0) = 0.5 σ(+∞) → 1
Formula 02
Forward Pass
Linear combination → sigmoid → probability
z = X · w + b ŷ = σ(z) = σ(X · w + b)
Formula 03
Binary Cross-Entropy Loss
Penalises confident wrong predictions exponentially
L = -1/n · Σ [ y · log(ŷ) + (1-y) · log(1-ŷ) ] y=1, ŷ≈1 → loss ≈ 0 ✓ y=1, ŷ≈0 → loss → ∞ ✗
Formula 04
Gradients + Weight Update
Chain rule — log + sigmoid derivatives cancel cleanly
∂L/∂w = X.T · (ŷ - y) / n ∂L/∂b = mean(ŷ - y) w ← w - lr · ∂L/∂w b ← b - lr · ∂L/∂b

Complete Pseudocode

LogisticRegression.pseudo
# Hyperparameters self.lr = learning_rate # step size, e.g. 0.01 self.epochs = epochs # max iterations, e.g. 1000 self.tolerance = tolerance # early stop threshold, e.g. 1e-4 # Learned parameters (set during fit) self.weights = None # shape: (n_features,) self.bias = None # scalar self.prev_loss = infinity
# Convert raw score → probability RETURN 1 / (1 + e^(-z)) # sigmoid(-∞) → 0 | sigmoid(0) = 0.5 | sigmoid(+∞) → 1
# y = actual labels (0 or 1) shape: (n,) # y_hat = predicted probabilities shape: (n,) loss = -mean( y * log(y_hat) + # term for class 1 (1-y) * log(1 - y_hat) # term for class 0 ) RETURN loss # single scalar — lower is better
── INITIALISE ────────────────────────── weights ← zeros(n_features) bias ← 0 ── TRAINING LOOP ─────────────────────── FOR epoch IN range(epochs): ┌── FORWARD PASS ────────────────────┐ z = X · weights + bias # linear step y_hat = sigmoid(z) # probabilities ┌── COMPUTE LOSS ────────────────────┐ loss = binary_cross_entropy(y, y_hat) ┌── EARLY STOPPING ──────────────────┐ IF |prev_loss - loss| < tolerance: BREAK prev_loss = loss ┌── COMPUTE GRADIENTS ───────────────┐ error = y_hat - y # (n,) grad_w = X.T · error / n # (features,) grad_b = mean(error) # scalar ┌── UPDATE WEIGHTS ──────────────────┐ weights = weights - lr · grad_w bias = bias - lr · grad_b
z = X · weights + bias probs = sigmoid(z) RETURN 1 if prob > threshold else 0 # per sample

Why Each Piece Exists

ComponentWhy it exists
sigmoid(z)Maps any real number to (0,1) — making the output interpretable as a probability. Neural networks need this.
X · w + bLearn a weighted combination of features. The bias b lets the decision boundary shift freely, not just pass through the origin.
binary_cross_entropyPenalises confident wrong predictions exponentially. Pairs perfectly with sigmoid — their derivatives cancel cleanly in the chain rule.
X.T in grad_wShape alignment. X is (n, f), error is (n,). X.T · error gives (f,) — exactly one gradient per weight.
(ŷ - y) errorLarge when wrong, small when right. Drives weight updates proportionally to how wrong the prediction was.
learning rateControls step size. Too large = overshoot. Too small = slow convergence. Typically 0.01 or tuned via grid search.
early stoppingStop when loss improvement < tolerance. Prevents wasted computation and reduces overfitting.
weights = zerosSafe initialisation. Unlike deep nets, logistic regression doesn't suffer from symmetry-breaking issues with zero init.

Connection to Neural Networks

Logistic regression IS a neural network — one neuron, no hidden layers. Everything here scales directly to deep learning.

sigmoid(z)→activation function (ReLU, tanh, softmax...)

binary_cross_entropy→loss function (cross-entropy, MSE, focal...)
X.T · (ŷ-y) / n→backpropagation (chain rule across all layers)

one neuron→millions of neurons across many layers
early stopping→early stopping + learning rate scheduler

Key Explanations

On Sigmoid
Sigmoid squashes any real number into a probability between 0 and 1. It's the activation function that turns logistic regression into a classifier rather than a regressor.
On Binary Cross-Entropy
Binary cross-entropy penalises confident wrong predictions exponentially. It's paired with sigmoid because their derivatives cancel cleanly, giving the simple (ŷ - y) error term.
On the Gradient Formula
The gradient X.T · (ŷ - y) / n comes from applying the chain rule to the loss. X.T is needed purely for shape alignment — one gradient per feature, not per sample.
On Early Stopping
Instead of blindly running all epochs, we track the loss each iteration and stop when the improvement drops below a tolerance threshold. Saves compute and reduces overfitting.
On the Neural Network Connection
Logistic regression is a single-neuron neural network. The forward pass is one linear layer + sigmoid. Backprop here is the chain rule across one layer — the simplest possible case.
